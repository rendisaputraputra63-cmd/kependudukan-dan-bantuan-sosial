pada halaman ini menampilkan beberapa fitur seperti : dashboard, data kependudukan,data kondisi rumah,klasifikasi penduduk,klasifikasi bantuan,control panel
Dashboard: Menampilkan statistik cepat dalam bentuk kartu dan grafik. Terdapat data jumlah penduduk berdasarkan jenis kelamin (Pria: 21, Wanita: 9, Total: 30) , serta grafik bantuan seperti BPNT, PKH, dan BST.
Daftar Data Kependudukan: Menampilkan tabel detail warga yang mencakup No KK, NIK, Nama, Hubungan Keluarga, Pekerjaan, hingga Penghasilan per Bulan.

ini adalah tampilan Halaman Data Kependudukan pada sistem SIMKBS (Sistem Informasi Manajemen Kependudukan dan Bantuan Sosial). Halaman ini dikelola oleh Admin dan berfungsi untuk menyimpan serta menampilkan database warga Desa Margodadi.


Halaman data kondisi rumah
<img width="1913" height="1016" alt="Screenshot 2025-10-28 153448" src="https://github.com/user-attachments/assets/4af76128-c7a7-4b66-a229-0a3377b1dc2b" />


Halaman ini digunakan untuk mendokumentasikan kelayakan hunian warga, yang biasanya menjadi indikator penentu bantuan sosial yaitu :
Kriteria Fisik: Mencatat Luas Lantai (contoh: <= 8m Persegi), Jenis Lantai (Semen/Tanah), dan Jenis Dinding (Bambu/Tembok Tanpa Plester).
Sanitasi dan Fasilitas: Mendata fasilitas MCK (Milik Umum), Sumber Penerangan (Tidak Menggunakan Listrik), Sumber Air Minum, dan Bahan Bakar Memasak.


Halaman kasifikasi penduduk
<img width="1917" height="1025" alt="Screenshot 2025-10-28 153502" src="https://github.com/user-attachments/assets/bae38e1a-ac59-4aa8-b18d-7d971f12ba68" />

Halaman ini menyajikan analisis statistik kependudukan dalam bentuk tabel yaitu :
Tabel Umur: Mengelompokkan warga berdasarkan rentang usia (0-5 hingga 61+) yang dipilah per dusun dan jenis kelamin.
Pendidikan: Menyajikan statistik warga berdasarkan Pendidikan Terakhir.


Halaman klasifikasi bantuan
<img width="1915" height="1012" alt="Screenshot 2025-10-28 153516" src="https://github.com/user-attachments/assets/92bc5882-c860-4bd5-b799-e612b542f600" />

Halaman khusus untuk mengelola daftar penerima bantuan sosial yaitu: 
Kategori Bantuan: Membagi data ke dalam tab bantuan BPNT, PKH, BST, dan BLT-Dana Desa.
Penetapan Penerima: Terdapat daftar warga beserta penghasilannya dan tombol + Tambah Bantuan untuk memasukkan warga tersebut ke dalam daftar penerima.
